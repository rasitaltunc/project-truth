# PROJECT TRUTH: COMPETITIVE LANDSCAPE ANALYSIS

**Research Date:** March 8, 2026
**Focus:** 3D network visualization + AI analysis + journalist protection + collaborative investigation

---

## EXECUTIVE SUMMARY

**Unique Position:** Project Truth occupies a **novel intersection** that no existing tool fully covers:
- **3D interactive network visualization** (Three.js/R3F engine)
- **AI-powered contextual analysis** (Groq LLaMA)
- **Journalist protection infrastructure** (Dead Man's Switch + Collective Shield)
- **Collaborative investigation system** (investigations, evidence staking, peer voting)
- **Epistemological evidence layers** (confidence tiers, source provenance)
- **Open-source architecture** (AGPL-3.0)

While individual components exist in separate tools, **no single platform combines all five simultaneously**. This is Truth's primary competitive advantage.

## SECTION 7: WHAT TRUTH DOES UNIQUELY

### 7.1 The 3D + AI + Protection Trinity

**No other tool combines:**

1. **Immersive 3D network exploration** (Three.js/R3F)
   - Users *enter* the network, rather than view it
   - Camera path through relationships
   - Zooming to node detail + glow rings
   - "Corridor" evidence timeline in 3D space
   - Tier-based orbital positioning

2. **Conversational AI analysis** (Groq LLaMA)
   - Chat with the network ("Who funds this network?")
   - AI highlights relevant nodes in real-time
   - Context-aware suggestions
   - Gap analysis ("What are we missing?")
   - Daily question generation

3. **Built-in journalist protection** (DMS + Collective Shield)
   - Individual Dead Man's Switch
   - Collective Shamir Secret Sharing
   - Proof-of-life blockchain check-ins
   - Graduated alert protocol (48h → 72h → trigger)
   - Geographic risk scoring

→ **Closest competitor combo:** Datashare (docs) + SecureDrop (sources) + Gephi (viz) = **3 separate tools, no integration**

---

### 7.2 Evidence Epistemology Layer (Unique)

Truth's advantage: **Confidence scoring built into the graph itself**

- **Evidence types:** court_record, official_document, leaked_document, interview, analysis, rumor
- **Confidence levels:** 0.00–1.00 (linked to visual properties)
- **Source hierarchy:** primary, secondary, tertiary
- **Evidence count:** visual link thickness
- **Verification badges:** official, journalist, community, unverified
- **Consensus annotations:** 10+ same label → persistent badge

→ **Competitors don't have this:**
- Maltego/Gotham: confidence hidden
- Gephi: attribute-agnostic
- Datashare: metadata-only
- Neo4j: user-defined labels (not standardized)

---

### 7.3 Collaborative Investigation Framework (Unique)

Truth features no competitor has:

- **Investigation objects:** Users propose connections ("I think X funds Y")
- **Evidence staking:** Reputation-based submissions (Tier 2+)
- **Community voting:** Up/down with tier weights
- **Auto-threshold acceptance:** 70% consensus → link added
- **Proposal lifetime:** 30 days + archive
- **Dual evidence system:** Confirmed links + proposed links (ghost edges)

→ **Closest alternatives:**
- ICIJ Datashare: document collaboration only
- Aleph: search + entity cards, no graphing
- Wikileaks: document dumps (no structure)

---

### 7.4 Epistemological Lens System (Unique)

5 different "investigation modes" through same network:

1. **Full Network:** All nodes/links visible
2. **Main Story:** Densest subgraph (story spine)
3. **Follow Money:** Financial nodes only
4. **Evidence Map:** Heat-mapped by query count
5. **Timeline:** Year-range slider, Z-axis ghosting

+ **Proposed mode: Board** (2D investigation canvas with sticky notes)

→ **Competitors:** Single view mode (maybe 2D/3D toggle)

---

### 7.5 Integration with Modern Journalism Workflow

Truth fills the gap between:

```
SecureDrop (source intake)
    ↓
OnionShare (file transfer)
    ↓
Truth (ANALYSIS + PROTECTION)
         ↓
Flourish (visualization for article)
         ↓
Publication
```

→ **No competitor owns the middle layer** (analysis + protection together)

## SECTION 10: RESEARCH FINDINGS - KEY INSIGHTS

### 10.1 The "No Combo" Problem

Research reveals: **60+ tools exist for investigative journalism, but none combine:**
- 3D interactive exploration
- AI conversation
- Journalist protection
- Collaborative investigation
- Evidence epistemology

This is Truth's **uncontested market**.

### 10.2 Proven Journalism Impact

ICIJ + Bellingcat + ProPublica have tested every tool category:
- Datashare (documents): "essential"
- Linkurious (visualization): "indispensable" (Panama Papers)
- Gephi (layout): "standard academic"
- SecureDrop (intake): "gold standard"
- Maltego (OSINT): "commercial standard"

**But none said:** "We want 3D + AI + Protection in one place"

**Reason:** Didn't exist as possibility until now

### 10.3 Open-Source Trust

Journalism community strongly prefers open-source:
- Bellingcat toolkit: 100% open-source emphasis
- Datashare: ICIJ's flagship
- Gephi: Research gold standard
- SpiderFoot: Preferred over Maltego for some orgs

→ **Truth's AGPL choice is competitive advantage**, not limitation

### 10.4 Journalist Safety is Underfunded

SecureDrop (intake) + Briar (messaging) exist, but:
- No protection *during* investigation
- No collective safeguards
- No "if something happens to me" protocols

→ **Dead Man's Switch + Collective Shield = novel feature class**

## CONCLUSION

**Project Truth occupies a unique, uncontested position in the investigative journalism technology landscape.**

While competitors own individual pieces (Gephi: visualization, Datashare: documents, SecureDrop: intake, Maltego: OSINT), no existing tool combines:

1. **Immersive 3D network exploration** with
2. **Conversational AI analysis** with
3. **Journalist protection infrastructure** with
4. **Collaborative investigation workflows** with
5. **Evidence epistemology layers** with
6. **Open-source transparency**

This combination is **not an accident**; it emerged from the specific needs of investigative journalism, where:
- **Journalists need immersion** (understand complex networks)
- **Journalists need AI help** (process massive datasets)
- **Journalists need protection** (operate in hostile environments)
- **Journalists need collaboration** (team investigations)
- **Journalists need to prove sources** (evidence confidence)
- **Journalists need transparency** (audit-friendly code)

**By March 2026, Truth is the only platform addressing all six simultaneously.**

The competitive threat is not today's tools but the **possibility that ICIJ, Neo4j, or well-funded startups will recognize this gap and fill it**. The window for Truth's adoption advantage is 6-18 months.
