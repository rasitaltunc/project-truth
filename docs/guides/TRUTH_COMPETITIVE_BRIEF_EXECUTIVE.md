## COMPETITIVE ADVANTAGES

### 1. Immersive 3D Exploration
**Truth:** Users *enter* the network via WebGL (Three.js/R3F)
**Competitors:** 2D link graphs (Gephi, Neo4j Bloom)

**Journalist benefit:** Understand complex networks intuitively; find "blind spots" in criminal networks visually.

### 2. Conversational AI
**Truth:** "Who funds this network?" → AI highlights relevant nodes + suggests connections (Groq LLaMA)
**Competitors:** Static visualization, manual pattern-finding

**Journalist benefit:** Reduce analysis time from weeks to hours.

### 3. Journalist Protection (Built-In)
**Truth:**
- Individual Dead Man's Switch (if jailed → evidence releases)
- Collective Shield (Shamir's Secret Sharing + proof-of-life blockchain)
- Graduated alert protocol (48h → 72h → trigger)

**Competitors:**
- SecureDrop: Source intake only
- Briar: Messaging only
- None address "what if the investigator is kidnapped"

**Journalist benefit:** Work safely in 50+ highest-risk countries (RSF index); platform automatically releases evidence if something happens.

### 4. Evidence Epistemology (Unique)
**Truth:** Links display confidence visually (color, glow, thickness)
- Confidence levels: 0.00–1.00
- Evidence types: court_record, leaked_document, interview, rumor
- Source hierarchy: primary, secondary, tertiary

**Competitors:** Attribute-agnostic (links have no meaning to system)

**Journalist benefit:** Transparent audit trail; readers see "this connection has 3 leaked documents + 1 court filing = 85% confidence"

### 5. Collaborative Investigation
**Truth:**
- Propose new links ("I think X funds Y")
- Community votes (Tier 2+ weighted)
- Auto-accept at 70% consensus
- Evidence staking (reputation-based)

**Competitors:**
- Datashare: Document collaboration (not graph)
- No one has community voting on connections

**Journalist benefit:** Crowdsourced verification; prevents single-analyst bias.

### 6. Open-Source (AGPL-3.0)
**Truth:** Fully auditable; security researchers can review
**Competitors:**
- Maltego/Gotham/Flourish: Proprietary (journalists can't verify)
- Some open-source (Gephi, Datashare) but not integrated

**Journalist benefit:** Use in high-security contexts (no backdoors); contribute custom features.
