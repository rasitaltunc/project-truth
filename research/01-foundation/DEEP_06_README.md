## RECOMMENDATIONS SUMMARY

### Licensing
✓ **Use AGPL-3.0 for code** (legally sound, community-trusted)
✓ **Add RAIL for AI components** (permits journalism, restricts surveillance)
✓ **Use ODC-PDDL for data** (once mature; fully open)
✗ **Don't use SSPL/BSL** (community hostile, companies retreated)
✗ **Don't use Hippocratic** (unenforceable, not OSI-approved)

### Governance Evolution
- **Phase 1 (Now)**: Founder-led (Raşit) + Advisory Council
- **Phase 2 (2027-2028)**: Steering Committee + PMCs
- **Phase 3 (2029+)**: Foundation model with Board of Directors
- **Key principle**: Only add complexity when scale demands it

### Ethical Enforcement
- **Legal enforcement**: Impossible for ethics clauses (courts don't adjudicate)
- **Social enforcement**: Works (Mastodon proved it)
- **Technical enforcement**: Possible for AI components (RAIL + API keys)
- **Recommendation**: Be honest about limits; use combination approach

### Community Protection
- **Code of Conduct**: Mandatory (Contributor Covenant v2.1)
- **CLA**: Mandatory (Apache CLA model; no copyright assignment)
- **Ombudsman/Appeals**: Essential at Phase 2+
- **Moderation process**: Graduated responses (warning → temporary ban → permanent ban)

### Funding Strategy
- **Year 1 (2026)**: Grants ($50-150K) + founder's time (unpaid)
- **Year 2-3 (2027-2028)**: Grants (50%) + Open Collective (10%) + Consulting (40%)
- **Year 4+ (2029+)**: Diverse (SaaS 30% + data licensing 20% + grants 30% + donations 20%)
- **Never do**: Venture capital, surveillance capitalism, targeted advertising

That's how trust gets built.
